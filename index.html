Hello World

Welcome to my website for my "csc220-coding challenges" repository

CSC 220 is an advanced programming class I am taking / took in the spring of my sophomore year. It is a class focused on advanced programming skills in 4 different languages (Python, Javascript, Java and Julia) taught by R. Jordan Crouser.

Excerpt from Course Description:

"This course will serve as an opportunity to gain hands-on experience with multiple programming languages through a series of challenging programming exercises. We will explore various techniques and using multiple programming languages including python, JS, Java and Julia ... Students will also develop good working knowledge of version control, collaborative programming techniques, coding conventions, and documentation."

Source: Course Website (Available Here)

About the Projects:

Coding "Sprints" are 20-40 minute projects assigned at the end of lectures
Coding "Challenges" are week long projects assigned each week


I created this website so that users can easily view the projects I created

For example, this is a great place to view Javascript projects in their html format!

Letter Frequency

Objective of challenge:

build a function that takes in a collection of strings, like
“the”, “quick”, “brown”, “fox”, “jumped” “over”, “the”, “lazy”, “dog”
and returns an object pairing each letter with its frequency of occurrence in the input strings: {a:1, b:1, c:1, d:2, e:4,...}


Description of my solution:

I created a dictionary object that utilized the letter as the key and the value as the string.
Then I looped through each character of each inputted string in a function (using spread to take in arguments since we do not know how many strings there are), and added 1 to the value of the key in my dictionary (where the key is the letter of the word).
After updating my dictionary, I printed it by looping through each key, value pair.
Finally, I return the results and alert it!

Test Case:

The test case utilized is an array of strings containing 'apple','banana','coconut','mango','peach' and 'strawberry'.
